Concepto: “Caos cromático”.

He generado 3000 círculos con posición, tamaño y color aleatorios (opacidad baja).
Las posiciones y tamaños se eligen con random(); los tamaños salen de {6, 10, 16, 24, 32}.
Visualmente representa orden + azar: un lienzo lleno de puntos de color sobre un fondo aleatorio.
Las variaciones se obtienen cambiando la semilla con las teclas **1**, **2** o **3** (101, 202, 303).
Para exportar cada versión, pulsa **S** (saveCanvas).
